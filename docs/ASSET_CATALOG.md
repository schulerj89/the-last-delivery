# Asset Catalog

This catalog documents the local source packs expected by the project. It is intentionally a source inventory guide, not a runtime manifest.

Run:

```sh
npm run assets:scan
```

The scan reports expected pack folders, tracked model file counts, file sizes, and missing folders. It does not copy files into `public/`.
