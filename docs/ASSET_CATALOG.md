# Asset Catalog

This catalog documents the local source packs expected by the project. It is intentionally a source inventory guide, not a runtime manifest.

Run:

```sh
npm run assets:scan
```

The scan reports expected pack folders, tracked model file counts, file sizes, and missing folders. It does not copy files into `public/`.

## Expected Source Packs

| Folder | Purpose | Runtime policy |
| --- | --- | --- |
| `raw-assets/low-poly-nature-pack-lite/` | Nature props and small environmental pieces | Select individual small props only |
| `raw-assets/fantasy-free-low-poly/` | Low-poly village/world props | Select individual small props only |
| `raw-assets/creative-characters-free/` | Character pieces and accessories | Source-only for now; do not import characters yet |
